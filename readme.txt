== Description ==

WP-RandomAds is Widget that will display random ads on your blog sidebar. You can put up to 6 ad code and ads will be displayed one at random. Visit <a href="http://wan.web.id" target="_blank">My Blog</a> For Detail Information.

== Installation ==

1. Upload 'WP-RandomAds.zip to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. See your widgets page
